# Getting Started with Next.js 14

Next.js 14 introduces the App Router, which provides a new way to structure your React applications...

## Key Features

- **Server Components** - Render components on the server by default
- **Streaming** - Progressive loading of UI components
- **Nested Layouts** - Share UI between routes while preserving state
